\## 📌 Project Overview

The \*\*CodeAlpha AI Assistant\*\* is a console-based Rule-Based Chatbot developed in Python as part of the \*\*CodeAlpha Python Programming Internship – Task 4\*\*.

Unlike a basic chatbot that only replies to a few fixed commands, this chatbot provides a more interactive experience using keyword matching, modular programming, random responses, and multiple conversation categories.

The chatbot is designed to simulate an AI-like conversation while remaining completely rule-based.
